# パパサポ

## 概要

出産前後1ヶ月でパパが情報を知ることができる
ママの心理状態や予定日1ヶ月前とかもわかる

## 前提

データはすべてフロントエンドで保持する

## ページ定義

### ページ1（出産予定日の設定）

- 出産予定日の入力
- 年月日を入力
- ページ2へ移行
- ローカルストレージに保存
  - ローカルストレージに値があればこのページは飛ばす
  - ローカルストレージに値がなければこのページに遷移する

### ページ2(出産前UI)

上から順番に記載

- 出産予定日まであと○日
- 赤ちゃんの画像（画像下に生まれたら赤ちゃんを長押ししてね）
- ママの状態
  - （心理状態とか身体の状態をドキュメントで記載）
- パパのTODOリスト
  - 5件のみ表示
  - さらに表示で読み込む
